## Overview

**CORTEXIA** is an intelligent healthcare platform designed to assist users with medical information, emergency services, disease intelligence, and AI-powered healthcare supportвЂ”all from a single interface.

It combines multiple AI modules with real-time health resources to create a smart healthcare ecosystem.

## Installation

Clone the repository

```bash
git clone https://github.com/your-username/CORTEXIA.git
```

Move into the project

```bash
cd CORTEXIA
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python -m streamlit run app\main.py
```
